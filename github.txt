important GITHUB commands
1.git add
moves changes from the working directory to the staging area.
2.git clone
Creates a copy of an existing Git repository. 
3.git fetch
Fetching downloads a branch from another repository
4.git reset
Undoes changes to files in the working directory.
5.git revert
can easily revert if we mistakely done a faulty commit.
6.git diff  
This command shows the file differences which are not yet staged.
7.git rm [file]  
This command deletes the file from your working directory and stages the deletion.
8.git branch [branch name]  
creates a new branch
